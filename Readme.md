‚öôÔ∏è Setup and Installation

Since this project is contained within a single .jsx file, setup is straightforward within a typical React environment.

Clone the Repository:

git clone https://github.com/Engineer-vejanand/vejanand-portfolio.git
cd vejanand-portfolio


Place the File: Place the portfolio.jsx content into your main React entry file (e.g., src/App.jsx or src/App.js).

Install Dependencies: Ensure you have Tailwind CSS and Lucide-React dependencies installed and configured in your project.

npm install tailwindcss lucide-react


Run Locally:

npm start


üìù Customization

All content is managed through the profileData object defined at the top of the portfolio.jsx file. To personalize the portfolio:

Update Contact Links: Modify the github and linkedin fields under profileData.contact with your actual profile URLs.

Add Project Links: For each entry in the profileData.projects array, update the demoLink (live site) and repoLink (GitHub code) fields.

Refine Content: Review and update the summary, skills, and project descriptions to ensure they accurately reflect your latest achievements and goals.
